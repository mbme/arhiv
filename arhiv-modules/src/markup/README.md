# Syntax
based on org-mode and markdown https://en.wikipedia.org/wiki/Lightweight_markup_language

* header `# Header`

`$ref = http://link | file-id`

* link `[[$ref][description]]`, description is optional

* bold `*text*`
* mono `` `text` ``
* striketrough `~text~`

* unordered list `* list item`

* code:
````
```js
// Code here
```
````

* blockquote:
````
```quote:source
Quote here
```
````

--------

### Later

* superscript `text^{1}`
* subscript `text_{2}`
* ordered list `- list item`
* line break (2 spaces at the end of the line)
* tables
* footnotes `^[note text]`
* italic `_text_`
